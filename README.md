# 📌 Top Projects

### 🩺 [ConvNeXt-V2 for Histopathology-Based Breast Cancer Metastasis Detection](https://github.com/jagadishpavanch/ConvNeXt-V2-for-Histopathology-Based-Breast-Cancer-Metastasis-Detection)

*Python • PyTorch • ConvNeXt V2 • Computer Vision • Medical AI*

> Histopathology images contain subtle microscopic features that make metastatic tissue difficult to identify consistently. This project evaluates **ConvNeXt V2** as a modern vision backbone for automated cancer detection, achieving **97.1% AUC** while outperforming ResNet-50, DenseNet-121 and EfficientNet baselines through transfer learning, advanced augmentation, and extensive hyperparameter optimization.
